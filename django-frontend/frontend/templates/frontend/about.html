About the library project

A Microservices, Containers, and Persistent Volume storage demo. This project contains a microservices application. The application is a very (very) simple web-based book lending app for a fictitious library. The project is intended to demonstrate a Microservices architecture within a Containerized solution using Docker Containers and Kubernetes orchestration. The "production" system will be deployed with persistent volumes utilizing the CSI driver, in order to demonstrate the ability to persist database data. {% endblock content %}
